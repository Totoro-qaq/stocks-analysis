# Code of Conduct

## Our Pledge

We pledge to make participation in this project professional, respectful, and harassment-free for everyone.

## Expected Behavior

- Be respectful of different viewpoints and experience levels.
- Give constructive feedback.
- Focus on technical issues and project goals.
- Avoid sharing private information, credentials, or restricted data.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing another person's private information without permission.
- Sharing secrets, credentials, proprietary data, or license-restricted datasets.
- Any conduct that would reasonably be inappropriate in a professional setting.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this code of conduct. Serious or repeated violations may result in a ban from project participation.
